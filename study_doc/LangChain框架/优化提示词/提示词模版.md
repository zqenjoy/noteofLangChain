# 提示词优化
 ## 通用prompt
    langchain提供了PromptTemplate类，构建可自定义的提示词模版，支持变量的注入，用来协助优化提示词。
    使用PromptTemplate.from_template()注入提示词变量占位
    使用链式写法定义变量的值，prompt_template.format(变量1=‘’，变量2=’‘)


## few shot的提示词
- 基于提示词的思想zero shot / few shot得到不同的提示词
- zeor shot可以使用通用prompt
- few shot则使用 FewShotPromptTemplate，使用链式语法创建提示词

### FewShotPromptTemplate模版语法
参数:
- examples:示例数据，list，内套字典.
- example_prompt:示例数据的提示词模板
- prefix:组装提示词，示例数据前内容
- suffix:组装提示词，示例数据后内容
- input_variables:列表，注入的变量列表

```
FewShotPromptTemplate(
    examples=None, 示例数据，list套字典
    example_prompt = None, 数据模版字符串
    prefix=None,            示例数据前的prompt
    suffix=None,            提问的prompt
    input_variables = []    声明在前缀或者后缀中所需要注入的变量名 list字符串
)
```
