# LangChain支持的模型

LangChain模型组件提供了与各种模型的集成，并为所有模型提供一个精简的统一接口。

- LLMs:是技术范畴的统称，指基于大参数量、海量文本训练的Transformer 架构模型，核心能力是理解和生成自然语言，
- 聊天模型:是应用范畴的细分，是专为对话场景优化的LLMs，核心能力是模拟人类对话的轮次交互，主要服务于聊天场景
- 文本嵌入模型:文本嵌入模型接收文本作为输入，得到文本的向量.

# 如何区分模型属于哪一类
    例如qwen-max 和 qwen3-max两种模型属于哪一类？

```commandline

llm = Tongyi(model='qwen-max')

print(type(llm))

chat = ChatTongyi(model='qwen3-max')

```

## LangChain community包

### LangChain支持对许多大语言模型的调用，以通义千问为例:

from langchain_community.llms.tongyi import Tongyi

### 实例化模型

llm = Tongyi (model='qwen-max')

### 模型推理

res=llm.invoke("帮我讲个笑话吧")
print(res)

### ollama为例:

from langchain_ollama import OllamaLLM
model= OllamaLLM (model="qwen3:4b")

### 通过invoke方法去调用模型

res = model.invoke(input="你是谁呀能做什么?")
print (res)
