# 了解提示词工程

## 什么是提示词、

在不更新模型权重的情况下，如何与大模型交互，以引导其行为获取所需结果的方法。
其实就是用户发出的指令

# 设计prompt的技巧

1. 详细的描述
2. 让模型充当一个角色
3. 使用分隔符表明输入的不同部分。比较中括号，三引号等分隔符划分要区分的文本，帮助模型更好的理解内容
4. 对人物指定步骤，拆分任务，明确任务步骤
5. 提供例子。
6. 使用参考文本作答，经典的方法是知识库用法，降低模型的幻觉问题

## 可用思路

### zero-shot

模型层面的理解：
模型可以使用训练过的数据特性，推广到集中的新类别上，也是零样本学习，解放（信任）预训练只是。
模型不知道要回答的问题，需要提供一些例子，让模型知道如何回答

提示词层面：
无提示，语言描述任务，依赖模型的预训练知识回答

### few-shot

模型层面：：
少样本呢，快速泛华识别新样本

prompt层面：给示例，引导模型对齐示例输出

# json数据格式

一种带有格式的字符串，便于不同程序之间的数据交互。
json.dumps(字典或列表,ensure_ascii=False)将字典转化为json字符串
json.loads(json字符串)将json字符串转化为字典或列表
