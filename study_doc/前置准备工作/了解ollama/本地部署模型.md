# 什么是ollama

Ollama 是一个开源的 LLM 模型，基于 LLama 模型，支持多个模型，如：LLama 7B、LLama 13B、LLama 30B、LLama 65B 等。
在ollama官网下载安装包[ollama官方模型库](https://ollama.com)

替代方案：qwen3 7b qwen13b qwen30b qwen65b，deepseekr1
